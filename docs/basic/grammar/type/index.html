    go struct 能不能比较？

    • 因为是强类型语言，所以不同类型的结构不能作比较，
    • 但是同一类型的实例值是可以比较的，
    • 实例不可以比较，因为是指针类型。

    类型转化

    Atoi

    // string 到 int
int, err := strconv.Atoi(string)
// string 到 int64
int64, err := strconv.ParseInt(string, 10, 64)

// int 到 string
string := strconv.Itoa(int)
// int64 到 string
string := strconv.FormatInt(int64, 10)
